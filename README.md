# BioAssay Studio：qPCR 与 Western blot 科研数据工作台

BioAssay Studio 是一个在浏览器本地运行的科研数据整理工具，包含 qPCR 相对定量、Western blot 灰度分析、内参归一化、上样量校准和论文图片排版。

## 在线使用

访问：<https://x738.github.io/bioassay-studio/>

首次成功打开后，网站会缓存程序外壳，断网时仍可继续打开。建议使用最新版 Chrome、Edge 或 Firefox。

## 主要功能

### qPCR ΔΔCt 分析

- 导入 CSV、TSV、XLSX 和 XLS 数据。
- 多工作表选择以及样本、基因、Ct/Cq 列匹配。
- Ct 范围、离群值和重复数据质控，可手动排除或修正。
- 设置一个或多个内参基因、对照组和生物学重复。
- 支持 ΔCt、ΔΔCt、`2^-ΔΔCt` 和扩增效率校正。
- 输出结果表、误差线图和 CSV。

### Western blot 灰度分析

- 单图自动推荐条带与背景 ROI，泳道数量不固定。
- 支持手动画框、移动、缩放、删除、改名和统一 ROI 尺寸。
- 提供全局平均、最近背景、左右局部背景和二维稳健背景平面四种扣除方式。
- 显示泳道灰度曲线、SNR、饱和比例和质量提示。
- 目的蛋白与内参双图按泳道配对，计算目的蛋白/内参和相对表达量。
- 可填写当前上样体积，并根据合格内参灰度估算建议上样体积。

### WB 图注与排版

- 多张 WB 图统一裁剪尺寸、紧凑黑框和水平校正。
- 支持蛋白名、分子量、泳道名、归一化值和分组标题。
- 字体使用 Times New Roman，并可分别调节字号。
- 自动定位有误时，可新增、移动、删除黄线并同步修改名称和数值。
- 导出带真实物理分辨率元数据的 600 DPI PNG、可编辑 SVG 或 PDF。

## 使用说明

1. 打开在线网站。
2. 选择“qPCR ΔΔCt 分析”或“WB 灰度分析”。
3. 从电脑选择实验文件；文件由浏览器直接读取，不会上传到本项目服务器。
4. 检查自动识别和质量提示，必要时使用手动编辑功能修正。
5. 导出 CSV、PNG、SVG、PDF，或保存项目 JSON 以便后续恢复。

## 计算说明

- `ΔCt = Mean Ct(target) - Mean Ct(reference)`
- `ΔΔCt = ΔCt(sample) - Mean ΔCt(control)`
- `Fold change = 2^(-ΔΔCt)`
- `WB 归一化 = 目的蛋白背景扣除积分灰度 / 内参背景扣除积分灰度`
- `建议上样体积 = 当前体积 × 基准泳道内参灰度 / 当前泳道内参灰度`

建议上样体积只适用于信号未饱和、灰度与上样量近似线性且样品浓度可比较的情况。

## 自动识别的适用边界

WB 自动识别采用非 AI、可解释的图像投影和局部峰值方法。它适合边缘裁剪较干净、泳道大致水平且没有严重过曝的图像。对弯曲泳道、拖尾、饱和曝光、拼图或文字标尺干扰，必须人工复核和修正。自动候选不能替代实验人员对条带身份和曝光线性范围的判断。
